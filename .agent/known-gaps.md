```txt
1. immutable runtime admission
```

## Required guarantees

```txt
required runtime sources are immutable and capability-admitted
reusable kit imports are side-effect free
one active runtime session owns all frames/listeners/resources
mission source is versioned, validated and fingerprinted
same seed creates the same objective entities
mission commands and contacts return typed results
progress and completion are deterministic and idempotent
render/HUD/GameHost consume one committed mission observation
restart creates a clean generation without stale mutation
terminal proof is bounded, detached and JSON-safe
```
