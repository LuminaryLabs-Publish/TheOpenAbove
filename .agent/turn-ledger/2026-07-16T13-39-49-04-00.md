## Main finding

A failed suite whose output matches the runner's broad assertion regex becomes a warning. Warnings do not block `npm run check`, and `npm run build` depends on that check. No explicit suite policy, finding ID, expected-drift record, waiver or expiry distinguishes stale assertions from real invariant failures.
